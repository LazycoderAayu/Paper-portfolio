## How to Contribute

### 1. Finding Layout Issues
- Found a section that looks broken on iPhone or wide monitors?
- Open an issue with a screenshot and the device/screen size details.

### 2. Feature & UI Requests
- Want to add a better scroll animation or a cleaner "Contact Me" form?
- Open an issue with the "UI/UX" tag.

### 3. Pull Requests 
1. Fork the repo.
2. Create a branch: git checkout -b fix/MobileResponsiveness
3. Commit your CSS/Logic: git commit -m 'Fixed overlapping text on mobile view'
4. Push it: git push origin fix/MobileResponsiveness
5. Open a Pull Request.

---

## Guidelines for Helpers
- Mobile First: Use media queries properly. Make sure it works on a 360px screen as well as 4K.
- Performance: Don't use massive images or heavy libraries that slow down the load time.
- Clean CSS: Use Flexbox/Grid. Avoid using "!important" unless necessary.
